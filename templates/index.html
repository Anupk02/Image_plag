Technology
How It Works
Text Analysis with Ratcliff Algorithm

Our system uses the Ratcliff/Obershelp pattern recognition algorithm to measure text similarity. This algorithm computes the similarity between two strings as:

similarity = (2 × number of matching characters) / (length of string1 + length of string2)

The algorithm finds the longest common substring, then recursively applies itself to the prefixes and suffixes of the strings for more accurate matching than simple string comparison.

Text Similarity Process
1. Extract text from PDF
2. Pre-process (clean text, normalize)
3. Compare with online text data using Ratcliff
4. Calculate similarity score
Image Matching Process
1. Extract images from PDF
2. Detect keypoints with ORB/SSIM
3. Compute descriptors
4. Match features across images
Image Matching with ORB+SSIM

For image plagiarism detection, we combine ORB (Oriented FAST and Rotated BRIEF) and SSIM (Structural Similarity Index Measure) algorithms:

ORB: Fast keypoint detection + rotation-aware descriptors
SSIM: works by comparing the luminance, contrast, and structure of two images to quantify their visual similarity on a scale from -1 to 1.

This hybrid approach provides both speed (from ORB) and accuracy (from SSIM), detecting manipulated or copied images even with transformations like rotation, scaling, or partial occlusion.
